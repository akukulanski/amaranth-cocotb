# nMigen-cocotb

nMigen-cocotb is a simple python module which tries to combine
two amazing HDL tools: nMigen and cocotb

# Installation

```console
pip3 install .
```

or

```console
pip3 install git+https://github.com/andresdemski/nmigen-cocotb.git
```
